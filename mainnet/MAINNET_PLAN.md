# SelfCoin Mainnet Plan (Skeleton)

## Principles
- Neutral money issuance by deterministic on-chain rules.
- No premine and no privileged admin keys.
- No consensus backdoors.

## Milestones
- [ ] Freeze mainnet parameters and publish checksums.
- [ ] Run validator ceremony and publish initial validator list.
- [ ] Publish canonical `genesis.json` and `genesis.bin`.
- [ ] Run multi-region dress rehearsal with public observers.
- [ ] Launch seed nodes and public lightservers.
- [ ] Declare mainnet start block height/time window.
